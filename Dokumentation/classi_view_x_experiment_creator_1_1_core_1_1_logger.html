Ausführliche Beschreibung

Dient als Preprocessor, vor Informationen an Serilog weitergegeben werden. So muss Serilog nicht überall referenziert werden und es erlaubt Reformatierung in einer zentralen Klasse. Damit sind keine Code-Schnipsel, die mit Logging-Reformatierungen zu tun haben, wild im Code verstreut. Außerdem werden hier das Log-Directory und Log-Files erstellt.
